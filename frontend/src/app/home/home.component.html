Features

Kangaroo is an objective tracking app focused on a simple UI to track tasks and deadline. It supports the ability to add subtasks and display progress for each subtask, as well as manually mark tasks as complete, incomplete or on hold.

About

Kangaroo is an open source project created by Faiq Ahmed, an undergraduate software engineering student at Mcmaster University with interests in DevOps and machine learning. This project was developed using angular, golang, postgresql, and nginx. The source code for the project can be found here.
